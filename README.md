Here is the **bigger‑step, action‑only checklist** you asked for. No concepts, no metaphors, no code.

---

## Project Setup

- Create a new project folder.
- Open the folder in your editor.
- Initialize npm.
- Install Express.
- Install nodemon as a dev dependency.
- Install TypeScript and its node/express type packages.
- Create a TypeScript configuration file.
- Set the project to use ES modules.
- Create a `src` folder for your TypeScript files.
- Create your main server file inside `src`.

---

## Data Setup

- Create a type that describes what a recipe looks like.
- Create an empty array typed as a list of recipes.
- Add a few sample recipes to the array.

---

## Server Setup

- Import Express using ES module syntax.
- Create an Express application instance.
- Enable JSON body parsing.
- Start the server on a port.

---

## Routes

- Add a route that returns the full recipe list.
- Add a route that returns one recipe based on an id from the URL.
- Add a route that accepts recipe details from the request body and adds a new recipe to the array.
- Add a route that updates an existing recipe based on an id from the URL.
- Add a route that removes a recipe based on an id from the URL.

---

## Testing

- Start the server.
- Test the route that returns all recipes.
- Test the route that returns one recipe by id.
- Test the route that adds a new recipe.
- Test the route that updates a recipe.
- Test the route that deletes a recipe.
- Re‑test the list to confirm changes.
